boforth - простой Forth интерпретатор

boforth - это минималистичный интерпретатор языка программирования Forth,
написанный на языке C. Проект следует принципам простоты и переносимости,
предлагая базовые возможности Forth для встраиваемых систем и образовательных целей.

## Требования

Для сборки boforth необходимы:

- компилятор C (gcc или clang)
- стандартные библиотеки C
- Just

Для разработки (необязательно):

- Nix
- gdb

## Установка

Готовые бинарники доступны в каталоге `./build`.

Для сборки из исходников выполните:
```bush
just release-native
```

При желании можно отредактировать `cfg.h` для настройки параметров.

## Запуск
```bash
    .build/release/boforth script.fs
```
